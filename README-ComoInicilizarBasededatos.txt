Pasos para ejecutar el programa:

1ºAbrir el xampp
2ºdesde el xampp iniciar apache y mysql, después acceder con el boton admin al localhost de phpmyadmin
3ºCrear en phpmyadmin una base de datos llamada "xijoja"
4ºAbrir el ejecutable del programa y darle en la rueda de ajustes de la parte de abajo, después darle a la opcion de resetear base de datos
5ºCon esto ya puedes usar el programa normalmente
6ºPara acceder desde el login estas serán las contraseñas y usuarios asignados por defecto:

Admin:         usuario=root, contraseña=root
Enfermero:     usuario=enfermero, contraseña=enfermero
Medico:        usuario=medico, contraseña=medico
Secretario:    usuario=secretario, contraseña=secretario
